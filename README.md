# MK Planner

Mini web app pour:
- entrer ton schedule du jour,
- gérer une box d’habitudes (cases à cocher),
- suivre la progression avec des barres,
- visualiser une `schedule bar` basée sur le temps planifié.

## Lancer

Ouvre simplement `index.html` dans ton navigateur.

## Fonctions

- Ajout de tâches avec heure de début/fin.
- Check des tâches terminées.
- `Progression schedule` (% de tâches finies).
- `Schedule bar (temps planifié)` (% de minutes finies + barre segmentée).
- Ajout d’habitudes avec checkbox.
- `Progression habitudes`.
- `Progression totale` (moyenne schedule + habitudes).
- Sauvegarde locale automatique (`localStorage`).
