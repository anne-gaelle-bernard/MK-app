# MK App

Welcome to MK App.

Stack:
- React frontend (Vite)
- JavaScript backend (Node.js + Express)

## Structure

- frontend: interface React
- backend: API JavaScript

## Installation

1. Installer les dépendances du frontend:
	npm --prefix frontend install

2. Installer les dépendances du backend:
	npm --prefix backend install

## Lancer en développement

Terminal 1:
npm run dev:backend

Terminal 2:
npm run dev:frontend

Frontend: http://localhost:5173
Backend: http://localhost:3001/api/health

## Features

- Welcome screen: "Welcome to MK App"
- 4 color blocks dashboard (Midnight, Pearl Perfect, Noir, Ocean)
- Weekly Schedule editor (7 days)
- Habit tracker
- Project tracker
- Progress bars + schedule bar
- Mobile mode: tap a color block to open feature panel
